Wii U VC Injection – UWUVCI AIO

Written by NomadTax

UWUVCI AIO supports:

  • DS
  • GBA
  • GB
  • N64
  • SNES
  • NES
  • TG16
  • MSX
  • Wii
  • GameCube

Compatibility Lists

  • DS: https://uwuvci.net/ndscompat
  • GBA / GB: https://uwuvci.net/gbacompat
  • N64: https://uwuvci.net/n64compat
  • SNES: https://uwuvci.net/snescompat
  • NES: https://uwuvci.net/nescompat
  • TG16: https://uwuvci.net/tg16compat
  • MSX: https://uwuvci.net/msxcompat
  • Wii: https://uwuvci.net/wiicompat
  • GameCube: https://wiki.gbatemp.net/wiki/Nintendont_Compatibility_List

Game Boy Note

  • Game Boy games do not have a separate compatibility list
  • Search the GBA compatibility list for your GB game of choice (e.g. Pokemon Gold)
  • Inject using the GBA tab in UWUVCI AIO

Using Compatibility Lists to Find the Base

  • Before injecting a game, check its compatibility list first
  • Find your game on the list to see which base game works for injection
  • Always read any notes or special instructions listed for that game
  • Use the recommended base in UWUVCI AIO to avoid crashes or errors

Wii U Common Keys (CKeys)

These Common Keys may work:

  • D7B00402659BA2ABD2CB0DB27FA2B656
  • 36262B5F49C69164E3BE2BB87C9922A7
  • A851D78AB8F0A6FE1E93CFCEAF99A179
  • Your personal CKeys can be obtained from otp.bin – simply press "Obtain Ckeys From otp.bin" when asked in UWUVCI AIO
  • Dump otp.bin using a NAND dumper to get your own keys
  • It is strongly recommended to use your own keys from otp.bin to avoid errors, crashes, or potential bricking

Wii U Title Keys (TKeys)

  • Title key database: https://phunlabs.github.io/
  • Title keys depend on the base game you select

Example (Ocarina of Time VC)

  • Use Donkey Kong 64 as the base
  • Search Donkey Kong 64 on the site
  • DO NOT use the "Copy Key" button
  • Copy ONLY the Title Key on the right

DK64 Title Key: 393396529b92ab77eb24302996bd4695
